# Diabetes Prediction using Machine Learning

## Overview

This project predicts whether a patient has diabetes using Support Vector Machine (SVM) trained on the PIMA Indians Diabetes Dataset.

---

## Features

- Exploratory Data Analysis (EDA)
- Data Preprocessing
- Feature Scaling using StandardScaler
- Support Vector Machine (SVM)
- Model Evaluation
- Real-Time Prediction
- Model Serialization using Joblib

---

## Technologies Used

- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Scikit-Learn
- Joblib

---

## Dataset

PIMA Indians Diabetes Dataset

768 Records

8 Clinical Features

Binary Classification

---

## Model Performance

Test Accuracy

76.6%
